Projects

2D Convolution

Repo on Github

Multiplied a 3-by-3 matrix kernel with an image and store the sum to each pixel to create the 2D convolution image processing effect.

Optimized the algorithm with a combination of C, SSE (Streaming SIMD Extentions), OpenMP, and cache blocking to achieve over 60 billion instructions per second (Gflop/s), a huge increase from the original 1 Gflop/s.

Hadoop MapReduce

Repo on Github

Learned and implemented MapReduce in Java using the Hadoop framework on the Amazon EC2 servers.

Developed 2 Mappers and 2 Reducers to calculate co-occurrence rate of words by finding the distance between target word and other words within a large document.

Sorted and outputted the results in descending order.

Hangman Solver

Repo on Github

Created an AI in Java to solve any Hangman game within 3 errors for Hulu's Programming Challenge.

Parsed mystery words from Hulu's designated website, then ran an algorithm with the alphabet frequencies of a large dictionary to determine the best guess.

Network

Repo on Github

Made a two player board game in Java, with the objective of forming a connection of chips in the same color without interference of opponent's chips.

Implemented a recursive Depth First Search to determine whether the current board contained a network.

We also utilized mini-max search algorithm and alpha-beta pruning to look for best possible move.

Mips Processor

Repo on Github

Constructed a CPU using Logisim that could read a subset of the Mips Instruction Set Architecture and process the instructions with pipelining.

I also build the Arithmetic Logic Unit and Controller to command the CPU.
